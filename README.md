# Farminfo Analytics Dashboard

이 프로젝트는 Farminfo의 판매 데이터(주문, 셀러, 상품 등)를 분석하고 시각화하기 위한 도구입니다. Streamlit 기반의 대화형 대시보드와 오프라인 분석 스크립트를 포함하고 있습니다.

## 📂 프로젝트 구조

주요 파일 및 폴더 설명은 다음과 같습니다:

- **`output/dashboard.py`**: 
    - **메인 대시보드 애플리케이션**입니다.
    - Streamlit으로 구현되어 있으며, 좌측 사이드바를 통해 5가지 분석 뷰(매출, 상품, 채널/지역, 고객, 셀러)를 제공합니다.
    - 배포 시 실행되는 진입점(Entry point)입니다.

- **`run_analysis.py`**:
    - **오프라인 분석 스크립트**입니다.
    - 대시보드를 실행하지 않고 로컬에서 분석을 수행하여 결과 그래프(이미지 파일)를 `output/` 폴더에 자동 저장합니다.
    - 정기 리포트 생성 용도로 활용할 수 있습니다.

- **`output/requirements.txt`**:
    - 프로젝트 실행에 필요한 Python 패키지 목록입니다 (`streamlit`, `pandas`, `plotly`, `python-dotenv`).
    - Streamlit Cloud 배포 시 이 파일이 참조됩니다.

- **`.streamlit/secrets.toml`**:
    - (로컬 환경 전용) Naver API 키 등 민감한 정보를 저장하는 폴더입니다.
    - **주의**: 이 폴더와 파일은 Git에 업로드되지 않도록 `.gitignore`에 포함해야 합니다.

- **`run_streamlit.sh`**:
    - 로컬에서 대시보드를 쉽게 실행하기 위한 쉘 스크립트입니다.

## 🚀 실행 방법

### 1. 대시보드 실행 (Dashboard)
터미널에서 다음 명령어를 실행하여 웹 대시보드를 켭니다.
```bash
./run_streamlit.sh
# 또는
streamlit run output/dashboard.py
```

### 2. 분석 스크립트 실행 (Script)
터미널에서 다음 명령어를 실행하여 분석 리포트(이미지)를 생성합니다.
```bash
python3 run_analysis.py
```

## 📊 주요 기능
대시보드는 다음 5가지 분석 탭을 제공합니다:
1. **매출 및 성과**: 일별 매출 추이 및 KPI.
2. **상품 분석**: 카테고리별 비중 및 인기 상품 랭킹.
3. **채널 및 지역**: 판매 채널 효율 및 지역별 매출 분포.
4. **고객 분석**: 재구매율, VIP 고객, 연령대별 고객 특성.
5. **셀러 분석**: 셀러별 매출 성과 및 신규 유입/이탈 추이.
